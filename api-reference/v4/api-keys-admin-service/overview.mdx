# API Keys Admin service overview

The API Keys Admin service provides team-wide, administrator-level operations for managing API keys across all team members in Coralogix. Unlike the [API Keys Service](../api-keys-service/overview), which scopes each call to an individual user's own keys, this service lets administrators list every API key issued to team members, bulk-delete keys, and bulk-activate or bulk-deactivate keys by status. Use it for offboarding flows, periodic key rotation, and team-wide audits. Access is governed by API-key admin permissions; see the [API Keys](https://coralogix.com/docs/user-guides/account-management/api-keys/api-keys/) user guide for the underlying key model and permission presets.

## Methods

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/aaa/api-keys/v3` | Retrieve all API keys for team members. |
| `POST` | `/aaa/api-keys/v3/all/delete` | Delete a list of specified API keys in bulk. |
| `POST` | `/aaa/api-keys/v3/all/status` | Activate or deactivate a list of specified API keys in bulk. |

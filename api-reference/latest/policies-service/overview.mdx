# Policies service overview

Coralogix Quota Policies API enables teams to define, enforce, and manage resource usage limits across logs and traces. With full CRUD capabilities, testing, and batch operations, this API ensures observability data flows remain within cost and compliance boundaries. Teams can apply quotas at the company level, toggle and reorder policies, and perform atomic updates for safe bulk changes.

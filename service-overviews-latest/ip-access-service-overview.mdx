# IP Access service overview

The Coralogix IP Access service provides comprehensive network-based access control for your observability platform, enabling organizations to enforce security policies by restricting access to specific IP addresses and ranges. Just like an Access Control List (ACL) for Coralogix. This service allows administrators to create, manage, and monitor IP access rules while maintaining granular control over Coralogix customer support access permissions. With full CRUD operations, bulk creation, and CIDR notation support, IP Access service ensures your observability data remains secure and compliant with organizational network security policies.
